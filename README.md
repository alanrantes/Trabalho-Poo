# Sistema de Logística

`CURSO: Análise e Desenvolvimento de Sistemas`

`DISCIPLINA: Programação Orientada a Objetos`

`4º semestre/2025`

<p align="justify">O projeto apresenta o desenvolvimento de um sistema orientado a objetos voltado para o gerenciamento de entregas em um ambiente logístico. A aplicação integra clientes, motoristas, veículos, endereços e entregas, permitindo registrar solicitações, atualizar status e acompanhar todo o processo de transporte. O objetivo principal é fornecer uma solução organizada, modular e de fácil manutenção, aplicando os conceitos fundamentais de POO, como herança, encapsulamento, polimorfismo e coesão.</p>
